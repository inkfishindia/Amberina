# Image assets for Amberina landing page

Add your real marketing images in this folder. The page will first try these primary names:

- `hero-estate.jpg` (fallbacks: `.jpeg`, `.png`, `.webp`)
- `estate-highlights.png` (fallbacks: `.jpg`, `.jpeg`, `.webp`)
- `tasting-notes.jpg` (fallbacks: `.jpeg`, `.png`, `.webp`)

## Why images might not show

- Wrong filename (must match exactly, including hyphen placement)
- Wrong extension (`.jpg` vs `.jpeg`)
- Image not committed to Git yet

If no expected file is found, the site now shows an inline placeholder image so the layout does not break.
