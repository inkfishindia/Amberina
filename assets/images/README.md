# Image assets for Amberina landing page

Add your real marketing images in this folder with the exact file names below:

- `hero-estate.jpg` — hero background image (estate panorama)
- `estate-highlights.png` — 3-icon estate highlights strip
- `tasting-notes.jpg` — tasting notes product/lifestyle image

Once these files are added and committed to Git, the website will use local assets directly (no external image host required).
